¿Cómo utilizar el kit medidor?

¿Qué es el kit medidor?

El kit medidor es un muestrario de tips reutilizables con diferentes tamaños que te permite medir tus uñas naturales para crear un set personalizado perfecto para vos.

Paso a paso

1. Probá cada tip en tus uñas
Colocá cada tip sobre tu uña natural sin pegamento.
  • Debe cubrir de lado a lado (pared lateral a pared lateral)
  • No debe sobresalir ni quedar chico
2. Ajuste correcto
El tip debe encajar perfectamente sobre tu uña.
  • Si duda entre dos tamaños → elegí el más grande
  • Evita que el tip presione la piel
3. Anotá tus medidas
Registrá el número correspondiente a cada dedo.
  • Pulgar
  • Índice
  • Medio
  • Anular
  • Meñique
4. Repetí en ambas manos
Las medidas pueden variar entre mano derecha e izquierda.
  • Tomá ambas medidas
  • No asumas que son iguales
5. Usá tus medidas al comprar
Con estos números podemos crear tu set totalmente personalizado.
  • Mayor precisión = mejor ajuste
  • Mayor duración del set

Importante

  • Siempre medí sobre uña natural limpia
  • No uses tips forzados
  • Elegir bien el tamaño mejora duración y comodidad
